Notes:

  1. Moderation commands require a channel set with x;modchannel to send Ban/Kick/Softban/Unban/Warn logs.
  2. To use member join/leave logging, set a channel with x;memberchannel. A custom message can be set with x;membermsg.
  3. Use x;help to view a command list
  4. Visit my home server for more support, or for updates, or if you just want to hang out.
